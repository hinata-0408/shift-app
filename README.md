# レジシフト自動作成アプリ

大手家電量販店でのアルバイト中に、レジ担当シフトの手動割り当て業務を自動化するために自作・実運用したWebアプリケーション。

---

## 背景・動機

毎日の業務で、当日出勤しているスタッフのシフト時間・経験レベル・担当回数を考慮しながらレジ枠を手動で割り当てる作業が発生していた。この判断はルールベースで自動化できると考え、Djangoで実装した。現場での実運用を経て、スコアロジックの調整や操作性の改善を重ねた。

---

## 機能

- **シフト自動生成**：スタッフのレベル・出勤時間・担当回数をもとにスコアを計算し、各時間枠へ最適な担当者を割り当て
- **ワンタップ変更**：シフト枠をタップして候補者一覧から即時変更
- **Excelシフト取込**：既存の全体シフト表（.xlsx）からの一括インポート
- **編集履歴**：誰がいつ何を変更したかを記録
- **スタッフ管理**：部門・レベル別の一覧、一括レベル変更

## セットアップ

```bash
# 仮想環境の作成と有効化
python3 -m venv venv
source venv/bin/activate

# 依存パッケージのインストール
pip install -r requirements.txt

# 環境変数の設定
cp .env.example .env
# .env を編集して SECRET_KEY を設定

# DBの初期化
python manage.py migrate

# 管理ユーザーの作成
python manage.py createsuperuser

# 起動
python manage.py runserver
```

> **注意**: `.env` ファイルは `.gitignore` に含まれており、リポジトリには含まれません。

## 工夫した点

### スコアベースの割り当てロジック

単純な優先度順ではなく、複数の要素を組み合わせたスコアリングで「最適な担当者」を決定している。

- **レベル係数**：Lv1（最優先）〜 Lv4（原則避ける）で基礎スコアを変動
- **担当回数ペナルティ**：同日に複数回担当するほどスコアが下がり、負荷が分散
- **短時間出勤補正**：勤務時間が短いスタッフへの過剰割り当てを抑制
- **シフト種別補正**：早番・遅番の違いによる時間帯マッチング

詳細は `register/register_logic.py` を参照。

### 実運用を前提とした設計

- 出勤データがゼロの日、全員が同レベルの日、途中入力など、実際に発生した異常系を考慮
- Excelシフト表は手入力のため表記揺れが多く、パーサーで複数のシフトコード形式に対応
- iPadでの現場利用を想定し、モバイル・横画面レイアウトに対応
